# Modal Footer Dark Mode Fix (Issue #14074)\n\n## What does this do?\nDemonstrates a modal footer with a proper dark mode background override that adapts the neutral-50 background to a dark surface.\n\n## How is it used?\n```html\n<div class="ease-modal-footer">...</div>\n```\n\n## Why is it useful?\nWithout a dark mode override, the modal footer renders as bright light grey inside a dark modal, breaking the visual theme consistency.
